# Documentação — Astrovistorias

Documentação técnica e operacional do sistema de frente de caixa multi-unidade.

## Estrutura sugerida

Conforme o projeto crescer, esta pasta organizará:

- **Planos de execução** — cronogramas de sprints, manutenções, migrações
- **Runbooks** — procedimentos operacionais (login, reautenticar Tiny, restaurar backup)
- **Arquitetura** — visão geral do sistema, fluxo de dados, decisões técnicas
- **Incidentes** — registro de problemas em produção e como foram resolvidos
- **Credenciais & Acessos** — referências (SEM segredos) a onde ficam tokens, URLs, etc

## Sincronização com Notion

Estes arquivos são a fonte de verdade versionada. Quando necessário, copie o Markdown para o Notion (ele converte automaticamente títulos, listas e tabelas).

O Notion é útil para operação do dia-a-dia (checklists, marcações, colaboração). Este repositório mantém o histórico auditável.
